Html Basic's

Details:

  • All HTML documents must start with a document type declaration.(< !DOCTYPE html>).

    • (There is no space between < ! above (It's so it shows on page. You'll notice this in certain spots!).

  • The HTML document itself begins with < html> and ends with < /html>.
  • The < html> tag represents the root of an HTML document.
  • The visible part of the HTML document is between < body> and < /body>.
  • The < !DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly.
  • It must only appear once, at the top of the page (before any HTML tags).
  • The< !DOCTYPE> declaration is not case sensitive.
  • The < !DOCTYPE> declaration for HTML5 is Case sensitive!!

KEY:

    Header Tags (All Tags are wrapped in < > )

  1. H1 Tag = Header Tag size 1 Biggest.
  2. H2 Tag = Header Tag size 2 bigger.
  3. H3 Tag = Header Tag size 3 big.
  4. H4 Tag = Header Tag size 4 small.
  5. H5 Tag = Header Tag size 5 smaller.
  6. H6 Tag = Header Tag size 6 smallest.

    7. Paragraph Tag

  7. P Tag = Paragraph Tag One size.

    8. List Tags

  8. ol Tag = Ordered List Tag.
  9. ul Tag = Unordered List Tag.
  10. li Tag = List Item Tag.

    11. Style Tags

  11. < b> < /b> = Turns element bold.
  12. < u> < /u> = Underlines element.
  13. < i> < /i> = Italics element.
  14. < br> < /b> = Line break Tag/ Start on next line.

    15. Comments Tag

  15. < !-- Comment goes here! -- > = To make a Comment.

Example Header Tags:

(< h1> < /h1>) = Biggest Header

(< h2> < /h2>) = Bigger Header

(< h3> < /h3>) = Big Header

(< h4> < /h4>) = Small Header

(< h5> < /h5>) = Smaller Header
(< h6> < /h6>) = Smallest Header
