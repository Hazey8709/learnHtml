Html: Hyper Text Markup Language

What is HTML ?

Most Basic building block of Web Development. Allows you to change/add website content

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

Html is useful for:

  • Web Developers
  • Software Developers
  • Maketing Professionals
  • Sales
  • Business Owners
  • Freelancers

What is a Div ?

The div tag defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The div tag is easily styled by using the class or id attribute.


What is span ?

The span HTML element is a generic inline container for phrasing content, which does not inherently represent anything. It can be used to group elements for styling purposes (using the class or id attributes), or because they share attribute values, such as lang .
